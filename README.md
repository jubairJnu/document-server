
# Document Server






## Live URL

https://document-server-eight.vercel.app/


## Run Locally

Clone the project

```bash
  https://github.com/jubairJnu/document-server.git
```

Go to the project directory

```bash
  document-server
```

Install dependencies

```bash
  npm install
 
```



```bash
  npm run dev
```
